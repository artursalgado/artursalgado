## 🚀 Sobre Mim

Sou programador e empreendedor em formação, baseado em Braga.  
Enquanto termino a licenciatura em **Engenharia Informática** no IPVC, desenvolvo produtos reais e lidero uma startup tecnológica — a **NovusCoding**.

Acredito que boa tecnologia nasce da combinação entre visão estratégica e execução técnica rigorosa.

```text
💡 Interesse principal   →   Arquitetura de software & sistemas escaláveis
🎮 A aprender agora      →   Unity + C# avançado
🏗️  A construir           →   Produtos próprios com impacto real
🎓 Formação              →   Lic. Engenharia Informática — IPVC
```
